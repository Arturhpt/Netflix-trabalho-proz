Desenvolvemos em grupo o site Lilflix, inspirado na Netflix, com foco em catálogo de filmes.

Tecnologias:

HTML para estrutura
CSS para design e layout escuro
JavaScript para interações

Funcionalidades:

Barra de pesquisa de filmes
Filtros por gênero, ano, nota e país
Catálogo com capas organizadas em grade
Navegação entre páginas (Início, Filmes, Série)
Botões interativos

Objetivo:

Praticar front-end
Trabalhar em equipe
Criar interface parecida com plataformas reais

Resultado:

Site funcional
Layout organizado e responsivo
Melhor compreensão de HTML, CSS e JS
